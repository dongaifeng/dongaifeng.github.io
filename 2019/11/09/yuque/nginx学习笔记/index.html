nginx学习笔记

Author: Dongaifeng Date: 2019-11-09,  21:33:57

1 安装

Linux 系统下安装

nginx-1.12.2.zip win 版本
安装 nginx 需要先安装几个插件
yum install gcc openssl openssl-devel pcre pcre-devel zlib zlib-devel -y

2 启动

普通启动

切换到 nginx 的 sbin 目录下 执行： ./nginx

实际工作中 通过输入路径 启动

/usr/local/nginx/sbin/nginx -c /usr/local/nginx/conf/nginx.conf

查看启动进程

ps -ef|grep nginx
nginx 有两个进程 master 进程读取配置文件，并维护 woker 进程。
woker 进程对请求进行实际操作。

3 关闭 Nginx

优雅关闭

找出主进程 master 号 ps -ef|grep nginx
kill QUIT 主进程 ID  
优雅关闭会在处理完已进来的请求并返回数据后才关闭

快速关闭

kill -TERM 主进程 ID
快速关闭 就是直接关闭 不在执行任务

重启 nginx

./nginx -s reload

检查修改配置文件是否正确

/usr/local/nginx/sbin/nginx -c /usr/local/nginx/conf/nginx.conf -t

4 window 安装

根目录下 启动：start nginx
关闭： nginx -s stop

5 配置文件 nginx.conf

反向代理

设置 linux 对外开放端口

firewall-cmd –aa-port=8888/tcp –permanent  对外开放 8888 端口
 firewall-cmd-reload 重启
 firewall-cmd-list-all 查看
