全栈项目学习笔记

Author: Dongaifeng Date: 2020-09-08,  18:59:14

搭建项目

  1. nuxt 搭建： npx create-nuxt-app <项目名>
  2. egg 搭建：npm init egg –type=simple
  3. 添加 husky：npm install husky –save-dev。添加配置信息。 官网
  1. 安装 commit 提交规范库 官网

登录方式

session-cookie

  1. 用户名+密码注册，node 接收信息，把信息保存到 mongdb 里面。
  2. 用户登录，校验用户名的唯一性。
  3. 验证码：email，图形，手机短信。后端随机生成验证码，前端验证。
  4. node 接受信息去 mongdb 查询用户信息，用户验证成功。
  5. 生成并保存登录状态到 session，并把返回到前端的 cookie。以后请求都会携带 cookie。

不好的地方：如果后台服务器好几台，session 不好管理，可以设 session 服务器，用 redis。

jwt（json-web-token）

  • 后端用 用户信息+过期时间 生成 token，返还给前端。
  • 前端把它存到 localStorage 中，axios 设置请求头 header.token = ‘token’。
  • 每次请求携带 token。后端解密 token。
